BRAND / DESIGN SYSTEM
- Aesthetic: sober operator-tech
- Color palette: dominant white, structural black, cold gray
- Typography: Geist Sans + Inter
- Interface: clean, documentary, precise
- Motion: minimal
- Visual language: diagrams, structure, systems
- Copy style: concrete, verifiable, no hype

GENERAL GOAL
This should not feel like a generic AI startup website or a trendy SaaS landing page.
It should feel like a serious operational consulting studio focused on data, workflow clarity, and selective automation.

The page must communicate:
- rigor
- clarity
- execution capability
- business judgment
- restraint

VISUAL RULES
- White must dominate the page
- Use black to define hierarchy and structure
- Use cold grays for borders, secondary text, separators, and subtle surfaces
- No gradients
- No glow effects
- No glassmorphism
- No colorful accents
- No heavy shadows
- No overly rounded playful UI
- No visual noise
- No futuristic AI clichés

TYPOGRAPHY RULES
- Use Geist Sans for headings, navigation, labels, and interface structure
- Use Inter for body copy and longer text
- Strong hierarchy, generous spacing, large headings
- Keep text blocks readable and restrained
- Avoid decorative or expressive typography

LAYOUT RULES
- Use a modern editorial / systems-based layout
- Large spacing between sections
- Clear section rhythm
- Strong alignment
- Minimal but precise use of cards
- Prefer blocks, dividers, grids, and structured whitespace over decorative containers
- The page should feel designed, not templated

MOTION RULES
- Minimal motion only
- Subtle fade or slight reveal animations are acceptable
- No parallax
- No floating effects
- No exaggerated hover effects
- No attention-seeking animation

VISUAL CONTENT RULES
- Do not use stock-photo style people
- Do not use robots, glowing brains, neon circuits, or abstract AI illustrations
- Prefer simple diagrams, clean structural visuals, workflow-oriented compositions, subtle system mockups, and restrained abstract layout elements
- The visual language should support the ideas of data, process, structure, and systems
